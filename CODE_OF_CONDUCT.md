# 社区行为准则

本项目采用基于 Contributor Covenant 2.1 精简改写的社区行为准则。

## 我们希望看到的行为

- 以尊重、耐心、合作的方式交流
- 对初学者友好，愿意解释上下文和设计动机
- 聚焦技术问题本身，给出可操作、可验证的反馈
- 尊重不同的经验背景、表达方式和学习节奏

## 不被接受的行为

- 人身攻击、侮辱、嘲讽、骚扰
- 恶意引战、钓鱼、刷屏
- 未经允许公开他人隐私或敏感信息
- 在技术讨论中使用带有歧视性的表达
- 明知会破坏项目协作氛围的言行

## 适用范围

本行为准则适用于：

- GitHub Issues
- Pull Requests
- 代码评审讨论
- 与本项目相关的公开协作空间

## 维护者责任

维护者有权删除、编辑或拒绝不符合本准则的评论、提交、Issue 或 Pull Request，并会尽量说明原因。

## 举报方式

如果你遇到不当行为，请联系仓库维护者：

- GitHub 主页：[yinlin712](https://github.com/yinlin712)

举报时请尽量提供：

- 发生位置
- 相关链接或截图
- 简要经过

维护者会尽量在合理时间内响应，并根据情况采取提醒、限制互动或拒绝参与等措施。

# 毕设实施建议与论文提纲

## 一、课题名称

《基于 Vibe Coding 的编程助手 Agent 研究与设计》

## 二、可落地的系统功能范围

建议先把毕设做成一个“可运行原型 + 可论证实验”的结构，不要一开始追求完整商用能力。

推荐最小可行功能：

- 在 VS Code 中提供聊天式编程助手
- 读取当前打开文件的上下文
- 生成代码解释、优化建议或简单代码片段
- 展示虚拟角色状态变化
- 接入一个本地模型接口
- 展示 LoRA 微调前后的效果差异

# Virtual

`virtual/` 用来集中管理 Code Agent 的数字人运行时。

这个目录当前承担的职责有：

- 管理数字人预设清单
- 管理 Webview 侧的数字人配置读取逻辑
- 管理数字人状态桥接与事件协议
- 为后续接入表情、动作、TTS、口型驱动和外部 API 预留扩展入口

当前文件说明：

- `avatar-presets.json`
  - 数字人预设清单，定义可切换角色的名称与 VRM 资源地址。
- `client-config.ts`
  - 在 Webview 中读取插件端注入的数字人配置，并完成前端所需的规范化处理。
- `protocol.ts`
  - 定义数字人运行时共享的事件协议。
- `bridge.ts`
  - 维护数字人状态桥，并向运行时广播会话状态、表达状态和当前文件信息。

后续如果需要继续扩展，建议优先在这个目录下增加：

- `api.ts`
  - 数字人业务接口，例如角色元数据查询、TTS 配置或表情驱动接口。
- `motions/`
  - 动作、姿态或表情映射资源。
- `assets/`
  - 本地预览图、离线 VRM 或其他配套资源。
